# Library Management System (Java OOP)
A simple Library Management System built using **Java Object-Oriented Programming (OOP)**.  
This project manages books, users, and basic library operations through a console-based menu.

## Features
- View all available books  
- Issue a book  
- Return a book  
- Predefined users  
- Organized using multiple classes (Book, User, Library, Main)
 
## Class Overview
# Book
- Holds book ID, title, author, and issue status  
- Methods: issueBook(), returnBook()
# User
- Stores user ID and name  
# Library
- Manages book list  
- Displays books  
- Issues and returns books
# Purpose
This project demonstrates:
- Basic Java OOP  
- Class interactions  
- Simple real-world logic  
- Menu-based program design
